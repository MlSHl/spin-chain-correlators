# Spin Chain Correlators

Numerical experiments for 1D quantum spin/free-fermion chains.

This repository currently contains scripts for:

* computing Toeplitz-determinant correlation functions,
* solving for Fermi-energy-related quantities,
* fitting numerical correlation data to asymptotic formulas.

## Files

```text
toeplitz.py
binary_search_fermi_energy.py
```

## Requirements

```bash
pip install numpy scipy matplotlib
```

## Usage

```bash
python toeplitz.py
python binary_search_fermi_energy.py
```

## Status

Early-stage research code for exploring correlation functions in 1D quantum many-body systems.
